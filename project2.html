Battle Tank: A mini game made with Unreal Engine

Introduction

This is the 3rd mini game that I am making with Unreal Engine 4. I made the first game under the perspective of a programmer, every single logics in the game was produced via C++. For the second game I worked under the perspective of a designers, Blueprint was used for almost everything. For BattleTank I am aiming to make it both as a programmer and designer. The game logics, ticks, architecture... are be made with C++ classes. At the same these classes are also exposed to the blueprint system where the constructors, UI, Input and variables are initialized. Meanwhile I also pay attention to good programming pratices and choose what not to expose to the blueprint.

Progress:

This project is currently under progress. A detailed report will be made soon. You can check the source code which is updated regularly on Gihub: BattleTank Github BattleTank GitHub
